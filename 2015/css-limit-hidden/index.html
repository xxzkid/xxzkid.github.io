CSS 限制字数，超出隐藏

发表于2015-12-24|更新于2026-07-15|html-css
|浏览量:

CSS 不能按照中文“字数”精确截断文本，但可以根据元素宽度或行数隐藏溢出内容，并在结尾显示省略号。

单行文本显示省略号

三个关键属性缺一不可：

  • overflow: hidden：隐藏超出容器的内容。
  • white-space: nowrap：禁止文本自动换行。
  • text-overflow: ellipsis：在截断位置显示省略号。

元素还需要具有明确宽度或最大宽度。
