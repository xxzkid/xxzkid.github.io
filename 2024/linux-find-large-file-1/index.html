Linux 查找大文件

发表于2024-02-18|更新于2026-07-15|linux
|浏览量:

磁盘空间突然不足时，通常需要先定位占用空间较大的文件。Linux 可以组合使用 find、du 和 sort，快速找出指定目录中的大文件。

查找大于 800 MB 的文件

参数说明：

  • .：从当前目录开始递归查找。
  • -type f：只查找普通文件，不包含目录。
  • -size +800M：筛选大于 800 MB 的文件。
  • -print0 与 xargs -0：正确处理文件名中的空格、中文和特殊字符。
  • du -h：以 KB、MB、GB 等易读格式显示实际占用空间。
  • sort -hr：按照容量从大到小排序。
