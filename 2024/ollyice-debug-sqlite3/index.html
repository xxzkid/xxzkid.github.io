  2. 打开OllyICE

CTRL+G 输入sqlite3_open

可以看到特征码

接下来我们分析微信的dll

找到微信的安装目录 C:\Program Files (x86)\Tencent\WeChat

找到版本目录下面的WeChatWin.dll 拷贝一份到D:\debug\WeChatWin.dll

我们打开IDA
New -》 选择D:\debug\WeChatWin.dll

ALT + b

双击进入

可以看到跟sqlitetest一样的代码
