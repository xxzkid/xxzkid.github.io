常见错误

  • 进制必须在 2—36 之间。
  • 输入字符串不能包含当前进制不支持的字符。
  • 非法输入会抛出 NumberFormatException。
  • 需要转换大整数时，不要先解析为 int 或 long，否则可能在创建 BigInteger 前已经溢出。
